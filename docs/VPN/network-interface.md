# Network Interfaces

Many VPN clients will create a network interface on your device when they are actived.

## List of Network Interfaces for Common VPNs

| VPN         | Interface     |
|--------------|-----------|
| Private Internet Access | wgpia0      |
| NordVPN      | utun1, utun2, utun3, utun4  |
| Mullvad | wg-mullvad |
